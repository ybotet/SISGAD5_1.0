## Описание проекта

Микросервис обрабатывает бизнес-логику и данные модуля MP.

## Архитектура

Реализован на Node.js и Express, взаимодействует с базой данных для хранения информации.

## Технологии

- Node.js
- Express
- База данных (настраивается через переменные окружения)

## Структура проекта

- `src/` с контроллерами, моделями и маршрутами.
- `config/` для конфигурации.
- `package.json` содержит зависимости и скрипты.

## Установка и запуск

```bash
npm install
npm run dev   # разработка
npm run start # продакшн
```

Настройте переменные окружения для базы данных перед запуском.
