## Описание проекта

Микросервис отвечает за аутентификацию и управление пользователями.

## Архитектура

Построен на Node.js и Express, предоставляет REST API и взаимодействует с базой данных.

## Технологии

- Node.js
- Express
- База данных (настраивается через переменные окружения)

## Структура проекта

- `src/` с контроллерами, моделями и маршрутами.
- `config/` для конфигурации.
- `package.json` содержит зависимости и скрипты.

## Установка и запуск

```bash
npm install
npm run dev   # разработка
npm run start # продакшн
```

Настройте переменные окружения для базы данных и безопасности.
