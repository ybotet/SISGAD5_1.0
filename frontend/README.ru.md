# Фронтенд микросервис

Содержание

- Описание проекта
- Архитектура

- Структура проекта

- Тестирование
- Контрольные вопросы

## Описание проекта

Пользовательский интерфейс, построенный с помощью Vite и React (TypeScript).

## Архитектура

SPA-приложение, которое общается с бэкенд-микросервисами через API.

## Технологии

- React
- Vite
- TypeScript
- Tailwind CSS (если используется)

## Структура проекта

- `src/`: исходный код приложения.
- `public/`: статические ресурсы.
- `package.json`: зависимости и скрипты.

## Установка и запуск

```bash
npm install
npm run dev   # запуск сервера разработки
npm run build # сборка для продакшена
```
