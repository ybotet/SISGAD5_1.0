## Описание проекта

Шлюз, который маршрутизирует запросы к другим микросервисам и обрабатывает общие задачи (аутентификация, логирование).

## Архитектура

Работает на Node.js, принимает внешние запросы и перенаправляет их к соответствующим сервисам.

## Технологии

- Node.js
- Express (или другой фреймворк)

## Структура проекта

- `src/` с `server.js` точкой входа.
- `package.json` с зависимостями и скриптами.

## Установка и запуск

```bash
npm install
npm run dev
npm run start
```
